pyZefir documentation
=====================
**Release:** 0.4.52

**Date:** October 18, 2024

PyZefir is a package that is used for optimization of energy network problems.
It is a Python library that provides well structured data output, that can be easily read
thanks to its various functions, for example, generating graphs representing the model.
When given an input data, it tries to optimize the problem based on initial parameters and conditions set by the user.
